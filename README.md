# bg-css

dark theme background transition

[preview](https://zhangyu1818.github.io/bg-css/)

## install

```shell
npm i bg-css
```

## usage

```js
import 'bg-css/style.css';
```

## Customize

modify css variables

```css
--theme-bg-light: #fff;
--theme-bg-dark: #000;
--theme-bg-duration: 700ms;
--theme-bg-transition: ease-in-out;
```
